Challenge 1

Es werde Licht

  • Lasse die erste LED rot leuchten.
  • Achtung: Wir fangen bei 0 an zu zählen!
NEU:
Farbe, Farben
Beispiel: Farben.Rot -> Farbe
setzeFarbe(position: Zahl, farbe: Farbe)
Beispiel: setzeFarbe(0, Farben.Rot); Setzt die LED 0 auf die Farbe Rot.

Challenge 2

Aus eins wird zwei

  • Lasse die erste LED blau und die zweite LED grün leuchten.

Challenge 3

Warte mal...

  • Lasse die erste LED rot leuchten, warte eine Sekunde und lasse sie dann grün leuchten.
NEU:
warte(millisekunden: Zahl)
Beispiel: await warte(1000); Wartet eine Sekunde.

Challenge 4

So viel Licht!

  • Lasse alle LEDs (32) weiß leuchten.
NEU:
setzeFarben(farben: Farbe[])
Beispiel: setzeFarben([Farben.rot, Farben.blau]) Setzt die LED 0 und 1 auf rot bzw blau.

Challenge 5

Blink blink

  • Lasse einen Punkt 5 mal blinken. Eine Sekunde an, eine aus.

Challenge 6

Kurzes auf und ab

  • Lasse einen Punkt von LED 0 bis zu LED 4 laufen und zurück.

Challenge 7

So viel Licht!

  • Lasse einen Punkt das Band hoch und wieder runter laufen.
NEU:
zähle i von a bis b in Schritten von c
Beispiel: zähle i von 0 bis 10 in Schritten von 1` Führt die Anweisungen 11 mal aus, wobei i bei der ersten Ausführung den Wert 0 hat, bei der zweiten den Wert 1 usw.
